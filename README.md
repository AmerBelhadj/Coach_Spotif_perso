# Coach Diastasis — Programme 16 Semaines PWA

> Application web progressive pour coach sportive — Diastasis + Recomposition corporelle Basic Fit

## 🚀 Déploiement GitHub Pages

1. Créer un repo GitHub public (ex: `coach-diastasis`)
2. Uploader **tous les fichiers** à la racine du repo
3. **Settings → Pages → Source: Deploy from branch → main → / (root)**
4. URL: `https://[username].github.io/coach-diastasis/`

## 📱 Installation PWA mobile

**iPhone Safari :** Partager (📤) → "Sur l'écran d'accueil"  
**Android Chrome :** Menu (⋮) → "Installer l'application"

## ✨ Fonctionnalités

- 16 semaines complètes (4 phases × 5 séances/semaine)
- Module diastasis sécurité (alertes, interdictions, alternatives)
- Offline-first (IndexedDB + Service Worker)
- Set Logger avec timer de repos automatique
- Vidéos YouTube par exercice
- Tracker nutrition avec repas types
- Graphiques poids / mensurations / diastasis
- Export CSV / JSON
- Thème dark/light
